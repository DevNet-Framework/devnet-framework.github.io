Computed Properties

A computed property is a property that doesn't store a value directly. Instead, it uses a getter and a setter method to retrieve and set other properties indirectly.

Getter

A getter is a function bound to a computed property and returns a value when that property is looked up.

To use a getter for a computed property in DevNet, you need first to include the trait DevNet\System\Tweak in your class to enable this feature, then define the computed property as a method prefixed with get that returns a value. 

<?php

namespace Application\Models;

use DevNet\System\Tweak;

class Person
{
   use Tweak;

   public string $FirstName;
   public string $LastName;
   
   // Defining the getter computed property.
   public function get_FullName(): string
   {
      return $this->FirstName . ' ' . $this->LastName;
   }
}

And here how you can access the computed property: 

<?php

use Application\Models\Person;

$person = new Person();

$person->FirstName = "John";
$person->LastName = "Doe";

// Accessing the computed property.
print($person->FullName); // Output: John Doe

Setter

A setter is a function bound to a computed property and will be called when there is an attempt to set that property.

To use a setter for a computed property, you need to include the trait DevNet\System\Tweak in your class to enable this feature, then define the computed property as a method prefixed with set that has a value as a parameter. 

<?php

namespace Application\Models;

use DevNet\System\Tweak;

class Person
{
   use Tweak;

   public string $FirstName;
   public string $LastName;
   
   // Defining the setter computed property.
   public function set_FullName(string $value): void
   {
      $segments = explode(' ', $value, 2);
      $this->FirstName = $segments[0] ?? '';
      $this->LastName = $segments[1] ?? '';
   }
}

And here how to set he computed property: 

<?php
               
use Application\Models\Person;

$person = new Person();

// Setting the computed property.
$person->FullName = "John Doe";

print($person->FirstName); // Output: John
print($person->LastName); // Output: Doe
