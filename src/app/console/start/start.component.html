Console Application

This chapter shows how to create and run a DevNet console application.

Create Console Project

To create a DevNet console project, enter the command below in the Terminal, where the argument project-name is any name you give to your project: 

devnet new console --output project-name

Or, using the composer command-line: 

composer create-project devnet/console project-name

Project Structure

The DevNet console project has the following folder structure: 

console-project/
├── bin/
|   ├── apphost
|   ├── run.bat
|   └── run
├── src/
|   └── Program.php
├── vendor/
└── devnet.proj

Entry Point

By default, the entry point is the static method Application\Program::main(array $args), but you can change that in the project file devnet.proj 

<?php

namespace Application;

use DevNet\System\IO\Console;

class Program
{
   public static function main(array $args = [])
   {
      Console::writeLine("Hello World!");
   }
}

Run The Application

To run the console application, run the following command in the Terminal : 

devnet run

Or, launch the application bin file directly: 

./bin/run

The program will display a "Hello World!" message in the console. 

Hello World!
