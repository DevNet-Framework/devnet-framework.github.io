Console Input

Console input is the input data that is entered by the user through the console window, to interact with system programs or other console applications.

User input

The static method DevNet\System\IO\Console::readLine(), is used to read the user input data from the standard input stream.

In the following example, the program prompts the user to enter his name, then reads the user input data and prints it in a greeting message. 

<?php

namespace Application;

use DevNet\System\IO\Console;
use DevNet\System\IO\ConsoleColor;

class Program
{
   public static function main(array $args = [])
   {
      Console::writeLine("What is your name?");
      $name = Console::readLine();
      Console.WriteLine("Hello, " . name . "!");
   }
}
Input
What is your name?
John Doe
Output
Hello John Doe!

Command arguments

Command line arguments are values specified after the program name and passed to the static method Program::main(array $args) during the execution, They are used for setting up the console application when it is started.

Hereâ€™s an example of how to handle the command line arguments: 

<?php

namespace Application;

use DevNet\System\IO\Console;
use DevNet\System\IO\ConsoleColor;

class Program
{
   public static function main(array $args = [])
   {
      Console::writeLine("The command line arguments:");
      foreach ($args as $argument) {
         Console::writeLine($argument);
      }
   }
}
Input
devnet run arg1 arg2 arg3
Output
The command line arguments:
arg1
arg2
arg3
