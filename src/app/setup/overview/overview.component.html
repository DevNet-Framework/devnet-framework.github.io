Overview

DevNet framework is free open-source software, and cross-platform, built-in PHP with high-performance, for rapid application development, to build console applications, web applications, web APIs and microservices.

Architecture

The DevNet framework has a layered architecture, which can be represented by the following diagram, that shows the major components of the DevNet Framework in each layer.

The lowest layer of the system provides the fundamental functionality that all components rely on, while the subsequent layers contain more sophisticated services that build on the layer below.

Here a brief summary of the DevNet components:

  • Application: a software that carry out specific tasks and operations, which the end-users interact with.
  • CLI Tools: a command-line interface tools, for creating console applications and managing DevNet projects.
  • Web library: a collection of web library and components, that helps to create web applications and web APIs.
  • Entity ORM: an object-relational mapping system, that helps to manipulate databases in an object-oriented way.
  • Core library: a runtime library that provides the fundamental utility and functionality to all DevNet components.
