Installation

DevNet Framework has some software requirements in order to work, so make sure your system meets the software requirements before getting to the installation.

Software Requirements

  • PHP version 8.1 or higher
  • Composer version 2.0 or higher
  • You may need to install or activate PDO PHP Extension
Important: In order to run DevNet Framework globally in Linux operating system, including DevNet CLI you need to add composer/vendor/bin into the system environment variables in .bashrc file like this:
export PATH="$PATH:$HOME/.config/composer/vendor/bin"

Installing DevNet Framwork

It is recommended to Install DevNet Framework globally in your system, just one framework runs multiple projects unless your projects use different versions of the DevNet framework, which may cause conflict because composer doesn't support multi-versioning, in this case, you can solve this problem using docker container or installing the compatible version of DevNet framework locally inside its proper project.

To install the DevNet framework globally using composer run the following command line in the terminal: 

composer global require devnet/framework

Or local installation inside your project by running the following command line in the terminal: 

composer require devnet/framework

Using DevNet CLI

After installing the DevNet framework, you can use DevNet CLI to create and manage your projects, and for the first start you can run the devnet command with the option --help in your terminal, to show the help on how to use the devnet command-line tools, as shown below: 

devnet --help
Output
Description:
  DevNet command-line interface

Usage:
  devnet [command] [options]

Options:
  --help, -h     Show help for the given command-line
  --version, -v  Show version information

commands:
  add            Add template code to the project.
  migrate        Update the database to a specified migration.
  new            Create a new DevNet project
  run            Run a DevNet project

Run 'devnet [command] --help' for more information on a command.
